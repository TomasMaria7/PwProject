API Metereologia

Descrição

Esta é a uma API de previsão do tempo, que fornece acesso aos dados meteorológicos para várias cidades. Para aceder a informações precisas sobre o clima diário e previsões para os próximos 5 dias em uma variedade de cidades use os endpoints referidos em baixo. Pode facilmente consultar o ID de uma cidade e obter dados meteorológicos detalhados para planear as suas atividades de melhor forma.

1. Listar Cidades

Endpoint: http://a22206963.pythonanywhere.com/meteo/api/lista_de_cidades/

Descrição: Retorna a lista de cidades disponíveis na API.

2. Previsão Meteorológica de Hoje para uma Cidade

Endpoint: http://a22206963.pythonanywhere.com/meteo/api/previsao_hoje/<cidade_id>/

Descrição: Retorna a previsão meteorológica de hoje para uma cidade específica.

Parâmetros:

  • cidade_id (string): O ID da cidade desejada.

3. Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

Endpoint: http://a22206963.pythonanywhere.com/meteo/api/previsao_proximos_5_dias/<cidade_id>/

Descrição: Retorna a previsão meteorológica para os próximos 5 dias para uma cidade específica.

Parâmetros:

  • cidade_id (string): O ID da cidade desejada.

Exemplo de URL para obter a previsão meteorológica de hoje para Lisboa (ID da cidade: 1110600):

http://a22206963.pythonanywhere.com/meteo/api/previsao_hoje/1110600/

Exemplo de URL para obter a previsão meteorológica para os próximos 5 dias para Lisboa (ID da cidade: 1110600):

http://a22206963.pythonanywhere.com/meteo/api/previsao_proximos_5_dias/1110600/
